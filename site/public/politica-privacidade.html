Política de Privacidade

Última atualização: 2 de março de 2025

Esta Política de Privacidade descreve como o App Empréstimos ("aplicativo", "app"), desenvolvido e operado pela Age Controle ("nós", "nosso"), coleta, usa, armazena e protege as informações dos usuários.

1. Informações que Coletamos

O aplicativo pode coletar as seguintes categorias de dados:

  • Dados de identificação pessoal: nome completo, CPF, RG, data de nascimento, sexo, telefone celular e e-mail.
  • Dados de localização: coordenadas de GPS (latitude e longitude) para fins de controle de atividades e gestão de empréstimos, quando o usuário autorizar o acesso.
  • Dados de conta: login, senha (armazenada de forma criptografada) e preferências de uso.
  • Dados de dispositivo: informações técnicas do dispositivo para garantir o funcionamento adequado do aplicativo.
  • Dados de uso: informações sobre como o aplicativo é utilizado para melhorar nossos serviços.

2. Finalidade do Tratamento

Utilizamos os dados coletados para:

  • Prestar os serviços de gestão de empréstimos e controle financeiro.
  • Autenticar usuários e garantir a segurança da plataforma.
  • Registrar e acompanhar localização quando necessário para as atividades do negócio.
  • Enviar comunicações relacionadas aos serviços (ex.: lembretes de parcelas, confirmações).
  • Melhorar a experiência do usuário e o desempenho do aplicativo.
  • Cumprir obrigações legais e regulatórias.

3. Base Legal (LGPD)

O tratamento de dados pessoais é realizado com base em:

  • Execução de contrato: para fornecer os serviços contratados.
  • Legítimo interesse: para melhorar nossos serviços e garantir a segurança.
  • Consentimento: quando exigido por lei ou para finalidades específicas (ex.: localização em segundo plano).

4. Compartilhamento de Dados

Podemos compartilhar dados com:

  • Prestadores de serviço: empresas que nos auxiliam na operação (hospedagem, APIs de pagamento, mapas).
  • Google: utilizamos serviços como Google Maps para funcionalidades de localização.
  • Autoridades: quando exigido por lei ou ordem judicial.

Não vendemos dados pessoais a terceiros.

5. Armazenamento e Segurança

Os dados são armazenados em servidores seguros e tratados com medidas técnicas e organizacionais adequadas para proteger contra acesso não autorizado, alteração, divulgação ou destruição. Tokens e senhas são armazenados de forma criptografada.

6. Permissões do Aplicativo

O aplicativo pode solicitar as seguintes permissões no dispositivo:

  • Localização: para registro de atividades em campo e gestão de empréstimos.
  • Câmera: para captura de documentos quando necessário.
  • Armazenamento: para salvar arquivos e dados do aplicativo.
  • Internet: para comunicação com nossos servidores.
  • Notificações: para enviar lembretes e alertas relevantes.

O usuário pode revogar permissões nas configurações do dispositivo a qualquer momento.

7. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem direito a:

  • Confirmar a existência de tratamento de dados.
  • Acessar seus dados.
  • Corrigir dados incompletos ou desatualizados.
  • Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários.
  • Revogar o consentimento, quando aplicável.

Para exercer esses direitos, entre em contato conosco pelos canais indicados abaixo.

8. Retenção de Dados

Mantemos os dados pelo tempo necessário para cumprir as finalidades descritas nesta política e para atender a obrigações legais. Após o término do período de retenção, os dados são eliminados ou anonimizados.

9. Menores de Idade

O aplicativo não é destinado a menores de 18 anos. Não coletamos intencionalmente dados de menores. Se tomarmos conhecimento de que coletamos dados de menor, tomaremos medidas para excluí-los.

10. Alterações

Podemos atualizar esta Política de Privacidade periodicamente. Alterações significativas serão comunicadas por meio do aplicativo ou por e-mail. A data da última atualização será indicada no topo desta página.
